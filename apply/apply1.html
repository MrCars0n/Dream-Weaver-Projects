Web Site Development Steps, Easy as 1-2-3

  1. The first step in developing a new Web site is to define a local site. A local site is defined in the Site Setup dialog box.
  2. Once the new site is defined, a new HTML page is created and saved. The new page is typically the home page and the home page is named index.html.
  3. Once the new HTML or home page is created, text is added to the page and then formatted.
