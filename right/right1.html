Dreamweaver Views

Dreamweaver allows you to review your Web page design using the following four views:

  • Code View. Code view displays the HTML, CSS, and any other Web programming language used to design the Web page.
  • Split View. In Split view, the designer can view a Web page's code and design.
  • Design View. Design view displays the Web page elements and design.
  • Live View. Live view provides a view similar to viewing a Web page in a browser. It shows what a Web page will look like in a browser.

For more information about switching between views in the Dreamweaver Document window, visit Dreamweaver Help.
