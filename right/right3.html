Learn HTML
Want to learn more about HTML and how to use it? Learn some of the basic HTML codes here!

The basic code for an HTML5 page includes:

<!doctype html>

<html>

<head>

<title>Title</title>

</head>

<body>Text</body>

</html>

Common HTML codes include:

Heading 1: <h1>Place heading here</h1>

Paragraph: <p>Place paragraph text here</p>

Link: <a href="index.html">Link text here</a>

Image: <img src="image_name.jpg" width="200" height="300" />

Line break: <br>

Bold text: <b>Text here</b>
